### Installation
```bash
  # clone the repository on your computer
  $ git clone https://github.com/entools/tools.interface

  # install dependencies
  $ npm install

  # run dev mode
  $ npm run dev

  # build project
  $ npm run build

  # clear project
  $ npm run clear

  # run eslint
  $ npm run lint

  # launch
  $ npm start
```
